# ברוכים הבאים לעולם ה-WEB! 🌐

בשיעור הזה נלמד איך דפדפנים קוראים דפי אינטרנט. כל מה שאתם רואים באינטרנט מתחיל ב-HTML – שפת הסימנים של הרשת.

## היררכיית התגיות
דף HTML בנוי כמו בניין:
- `<html>` - המבנה כולו.
- `<body>` - התוכן שרואים בעיניים.
- `<h1>` - כותרת גדולה וחשובה.

### בואו נבנה את האתר הראשון שלכם:
נסו לשנות את הטקסט בתוך ה-H1 ולראות איך זה משתנה בתצוגה המקדימה מצד שמאל!

<CodeRunner 
  language="html"
  initialCode={`<!DOCTYPE html>

  </button>
